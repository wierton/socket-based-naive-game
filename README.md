* install
  ```shell
  git clone https://github.com/ycpedef/socket-stg
  # or
  # git clone https://gitee.com/yuanchenpu/socket-stg
  cd socket-stg
  make
  ```
* run  
  1. run `./server` in one terminal
  2. run `./client [server_ip]` in another terminal
